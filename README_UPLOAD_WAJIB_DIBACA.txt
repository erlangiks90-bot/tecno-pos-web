CARA PAKAI FILE INI:
1. Ekstrak ZIP ini.
2. Buka folder hasil ekstrak.
3. Copy SEMUA ISI folder ini ke folder repo GitHub tecno-pos-web.
4. Jangan upload ZIP langsung ke GitHub.
5. Jangan hapus folder .git di repo lama.
6. Jalankan:
   npm install
   git add .
   git commit -m "Fix full folder POS"
   git push origin main
