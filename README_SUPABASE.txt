TECNO POS SUPABASE READY

1. Buat project Supabase
2. Project Settings > Database > Connection string > URI
3. Copy URL, ganti [YOUR-PASSWORD] dengan password database
4. Railway > Variables > tambah DATABASE_URL=postgresql://...
5. Deploy ulang Railway
